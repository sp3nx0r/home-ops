| Role        | Hostname     | Notes |
| ----------- | ------------ | ----- |
| CP + worker | `miirym`     | Ghost dragon bound to guard Candlekeep’s library — a node that should stay up. |
| CP + worker | `palarandusk` | The Sun Dragon, quiet guardian — reliable workhorse. |
| CP + worker | `aurinax`    | Gold dragon guarding a hoard in the Vault of Dragons — workloads and state. |
